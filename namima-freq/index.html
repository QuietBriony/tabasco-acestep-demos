namima — frequency ambient (candidate)

同じオフライン numpy シンセで作った参考デモ。上=メロディのスケッチ、下=周波数ドローン。すべて自己合成・48kHz。
⚠ これは candidate(実験)の出力です。namima 本体アプリ(ビジュアル・アンビエント)とは別物で、 Tone.js の実行時合成ではなく オフライン生成。サンプルは一切使わず全部シンセ、生成物は製品 repo には入れていません。

⚡ Solfeggio IDM — Xtal / On mood (new)

「単調・音色がない」への返答。Aphex Twin Xtal / On の気分で、114BPM・~3:08。 周波数→音色の割り振り:174/285(低)=バウンスするオクターブ・ベース+ブレシーなパッド、 396/417/528(中)=カープラス・ストロング撥弦の合いの手+声の空気、 639/741/852/963(高)=FMマレット・ベルのメインリフ(Onのあの打音)。 リフは間(休符)込みの2小節フレーズが変化(ミュート/差し替え)しながら回り、 スウィング+ベロシティ+シンコペするベースがキックと噛んでグルーブ。 音程は全部ソルフェジオ絶対Hz(非ドレミ)・グライドなし・全部自己合成。
v2: ドラムはブレイク切り刻み——2小節ブレイクを合成→16分スライスに割って スタッター/逆再生/入替/半速/ミュートで組み替え(後半ほど激しく、8小節目にスネアロール)。 ベースは極太バス:100Hz以下のサブ(43.5/71.25/87/142.5Hz = ソルフェジオのオクターブ下、 非ドレミのまま)+ 中域レイヤーを軽くドライブで接着、キック同期のポンピングで体にくる低音。 🔊 サブはスピーカー/ヘッドホン推奨(電話の小スピーカーでは倍音でのみ聞こえます)。
⚡ “Solfeggio IDM” v2 — ~3:08 / 114BPM / 切り刻みブレイク + 極太サブ + FMベル・リフ / 非ドレミ

🌊 Solfeggio drift — ambient (v2・品質パス)

深いビート + たゆたうメロディの ~3分アンビエント(Alberto Balsalm の気分で)。 肝は音程:メロディも声も ソルフェジオ周波数そのもの(174/285/396/417/528/639/741/852Hz)を 音階に使い、ドレミ(12平均律)の外で鳴らす(396↔417 の 89セント微分音など)。 音程が滑るリードは廃止 → 柔らかい音が重なってクロスフェードする漂うライン。 888Hzベルと8888Hzのキラキラ(小音量)も織り込み済み。
v2 の品質パス(4視点レビュー→敵対検証で磨いた版): ①iPhoneスピーカーで鳴るキックの実体(中低域)を追加、 ②モノ互換(前版は9msのステレオでモノ合成時に中域が櫛状に欠けていた→L+R=2×midになる方式へ)、 ③ベル/キラキラのクリック除去、④メロディの音域の偏り解消、 ⑤パッドの明るさ推移+サイドチェイン、⑥声をより温かく。 自己合成・MP3 224kbps・🎧 or スピーカー。コードは namima repo に候補として統合済み。
