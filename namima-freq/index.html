namima — frequency ambient (candidate)

同じオフライン numpy シンセで作った参考デモ。上=メロディのスケッチ、下=周波数ドローン。すべて自己合成・48kHz。
⚠ これは candidate(実験)の出力です。namima 本体アプリ(ビジュアル・アンビエント)とは別物で、 Tone.js の実行時合成ではなく オフライン生成。サンプルは一切使わず全部シンセ、生成物は製品 repo には入れていません。

🌊 Solfeggio drift — ambient (new)

深いビート + たゆたうメロディの ~3分アンビエント(Alberto Balsalm の気分で)。 肝は音程:メロディも声も ソルフェジオ周波数そのもの(174/285/396/417/528/639/741/852Hz)を 音階に使い、ドレミ(12平均律)の外で鳴らしています(396↔417 の 89セント微分音など)。 音程が滑るリードは廃止 → 柔らかい音が重なってクロスフェードする漂うライン。 ベッドは温かいデチューン・パッド + 深いサブ、上に フォルマント合成のハミング(ノラ的な温度)。 全部この numpy シンセで自己合成、MP3 224kbps・🎧 推奨。
🌊 “Solfeggio drift” — ~3:04 / 深いビート + 漂うメロディ / 非ドレミ / 自己合成
🔔 “Solfeggio drift + 888/8888” — 同じ曲に 888Hzのベル(チャイム)と 8888Hzのキラキラを重ねた版
同じ曲に 888Hz(A5の少し上)を非整数倍音のベル/チャイムとしてフレーズ頭に、 8888Hz をまばらな小音量のキラキラ(空気)として散らしました。 8888Hz は持続させると刺さるので、柔らかい減衰の粒として控えめに(実測でも中域の約1/20の音量)。
